  Creating matrix with sigly-linked-list
  x and y are indexes of the matrix and val is the value of the matrix at that index
  Example: 
  
  x=0
  y=0
  val=0
  output: 20
 
  x=1
  y=2
  val= 15
  20 -> 0 -> 0
  |    |    |
  V    V    V
  0 -> 0 -> 15
